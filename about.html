About

What's the purpose of Safe Happy?

This server is made to help improve mental health and form a support system for everyone. You can give help, get help, vent, or just make some friends. Whatever you need, we are here for. We DO NOT tolerate hate words, phrases, or actions.

Community

This server acts as a community that supports those with mental illnesses. In this community, you can chat with other members that are experiencing the same thing you are. This community is meant to make its members feel safe and loved.

Support System

This server will also act as a support system for the mentally ill. The Hotlines channel provides emergency hotline numbers when a specific command is used. This is to help with any personal crisis or situation that comes up. We are here to support each other.

Happiness

One of this server's purposes is to give you happiness because you deserve it.

Kindness

You can count on Safe Happy to have kind members to help you through hard times.

Loving

Lastly, this server provides a safe space where, no matter what, you can be yourself and you will be loved.

The creator/moderator

Hello!! I'm the creator @Chaotic_Potato. I also moderate Safe Happy!! I created this Discord server to form a community and support system for the mentally ill. I hope to help others create a support system of friends to rely on.
